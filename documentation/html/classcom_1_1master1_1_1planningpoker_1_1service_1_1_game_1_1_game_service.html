Detailed Description

Service qui gère les opérations relatives aux jeux dans le système de planification de poker. Cette classe implémente l'interface IGameService pour créer, éditer, récupérer et supprimer des jeux.

Crée ou modifie un jeu selon les informations spécifiées dans la requête. Si un jeu existe déjà avec l'ID fourni, il sera mis à jour. Sinon, un nouveau jeu sera créé.

Parameters
requestLa requête contenant les informations nécessaires pour créer ou modifier un jeu.
Returns
Un objet GameResponse contenant les détails du jeu créé ou mis à jour.
Exceptions
IllegalArgumentExceptionsi le nombre maximal de joueurs est inférieur ou égal à zéro, ou si la règle spécifiée dans la requête n'existe pas.
