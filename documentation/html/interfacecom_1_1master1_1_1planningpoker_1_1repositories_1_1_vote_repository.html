Detailed Description

Interface du dépôt pour l'entité Vote, permettant l'accès aux données relatives aux votes des joueurs.

Cette interface étend JpaRepository pour fournir des méthodes de gestion des entités Vote dans la base de données. Elle permet de récupérer les votes par assignment ou par code de jeu.
