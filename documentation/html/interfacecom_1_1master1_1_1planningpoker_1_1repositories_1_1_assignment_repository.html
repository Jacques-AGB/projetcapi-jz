Detailed Description

Interface du dépôt pour l'entité Assignment, permettant l'accès aux données relatives aux assignations (tâches).

Cette interface étend JpaRepository pour fournir des méthodes de gestion des entités Assignment dans la base de données. Elle inclut des méthodes pour récupérer des assignations en fonction de différents critères, comme le libellé ou le code de jeu.
