Detailed Description

Gestion des requêtes HTTP liées aux votes.

Ce contrôleur permet de gérer les opérations sur les votes dans le système. Il fournit des fonctionnalités pour soumettre un vote, récupérer des votes pour un joueur, une tâche ou un jeu, ainsi que supprimer un vote.
