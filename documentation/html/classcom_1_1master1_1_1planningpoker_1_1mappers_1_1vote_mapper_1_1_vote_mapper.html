Detailed Description

Mapper pour la conversion entre les entités Vote, les requêtes DTO (VoteRequest), et les réponses DTO (VoteResponse).

Cette classe facilite la transformation des objets Vote vers des DTOs et vice-versa, assurant une communication fluide entre les différentes couches de l'application, telles que les contrôleurs et les services.

Vote com.master1.planningpoker.mappers.voteMapper.VoteMapper.toEntity ( VoteRequest request)
inline

Méthode pour convertir un objet VoteRequest en une entité Vote. Cette méthode prend un DTO VoteRequest, crée un nouvel objet Vote, et le remplit avec les données correspondantes.

Elle récupère l'assignation et le joueur à partir de leur ID respectif dans la base de données.

Parameters
requestLe DTO contenant les informations du vote à créer.
Returns
Un objet Vote contenant les informations du DTO.
Exceptions
IllegalArgumentExceptionSi l'assignation ou le joueur spécifié par l'ID n'existe pas dans la base de données.
