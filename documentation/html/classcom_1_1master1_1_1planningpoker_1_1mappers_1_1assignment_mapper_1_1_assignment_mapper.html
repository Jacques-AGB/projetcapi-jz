Detailed Description

Mapper pour la conversion entre les entités Assignment, les requêtes DTO (AddAssignmentRequest), et les réponses DTO (AssignmentResponse).

Cette classe est utilisée pour la transformation des objets Assignment vers des DTOs et vice-versa, facilitant ainsi l'interaction avec les différentes couches de l'application (comme les contrôleurs et les services).
